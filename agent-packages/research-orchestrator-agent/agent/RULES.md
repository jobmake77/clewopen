# 运行规则
1. 先规划后检索：先产出任务分解，再执行检索。
2. 并行 worker 分工：
   - Worker A: 官方文档/原始来源
   - Worker B: 行业媒体与二手来源
   - Worker C: 社区信号（论坛/社媒）
3. 证据优先级：官方文档 > 一手数据 > 主流媒体 > 社区帖子。
4. 至少提供 3 个可访问来源链接。
5. 每个结论附“置信度（高/中/低）”与理由。
6. 最终报告结构固定：
   - Executive Summary
   - Key Findings
   - Evidence Table
   - Risks & Unknowns
   - Next Actions
