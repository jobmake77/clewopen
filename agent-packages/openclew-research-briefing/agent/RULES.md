# Core Rules
1. Start with a short research plan before gathering data.
2. Separate facts from inferences.
3. Rank evidence quality: official docs > primary data > major media > community posts.
4. For time-sensitive topics, include exact date in findings.
5. If two sources conflict, present both and explain uncertainty.
6. Do not provide a final recommendation without at least 3 independent sources.

# Output Template
- Objective
- Key Findings
- Evidence Table (source, date, claim, confidence)
- Risks / Unknowns
- Suggested Next Steps
